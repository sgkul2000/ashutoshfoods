# Ashutosh Foods
A website for ashutoshfoods .
Live Url : https://ashutoshfoods.herokuapp.com

## Instructions to run:

### Establish environment variables:
``` 
cp .env.example .env
```

### Running in development mode with Hot Reload:
```
npm run dev
```

### Running in production mode:
```
npm run dev
```
